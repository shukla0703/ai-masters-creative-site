# 🌐 AI Masters Creative CV Website

## 📖 Project Overview

The **AI Masters Creative CV Website** is a modern and interactive personal portfolio designed to present my professional profile, technical skills, and project experience in a visually engaging way. It functions as a digital resume that allows recruiters, collaborators, and visitors to easily explore my background in data science, AI, and software development.

The website highlights my academic achievements, professional projects, and technical capabilities while providing a centralized platform for showcasing my work and career journey.

## ✨ Key Features

* Responsive and modern **portfolio-style CV website**
* Structured sections including **About, Skills, Projects, Experience, and Contact**
* Clean and intuitive **user interface**
* Interactive **project showcase**
* Integration with **GitHub and professional profiles**
* Fully optimized for **desktop and mobile devices**

## 🛠️ Technologies Used

* **HTML** – Structure and layout of the website
* **CSS** – Styling and responsive design
* **JavaScript** – Interactivity and dynamic behavior
* **Git & GitHub** – Version control and project hosting

## 🎯 Purpose of the Project

The purpose of this project is to create a **professional online presence** where my technical expertise, projects, and achievements can be showcased in a structured and accessible format. It serves as a personal portfolio to demonstrate my skills and attract potential employers, collaborators, and opportunities in the technology domain.
